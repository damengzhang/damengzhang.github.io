Axios是什么

Axios 是一个基于 promise 的 HTTP 库，可以在浏览器和 node.js 中使用。

Axios特点

  • 支持在 浏览器 和 node 中使用
  • 支持 Promise API
  • 拦截请求和响应
  • 转换请求数据和响应数据
  • 取消请求
  • 自动转换 JSON 数据
  • 客户端支持防御 XSRF
