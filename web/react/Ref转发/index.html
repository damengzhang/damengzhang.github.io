Ref转发

发表于 2019-04-09 分类于 web ， react

作用

Ref转发允许组件接收ref参数，并将ref向下转发给子组件，进而达到在组件外部对组件的子组件进行管理的目的。可用于需要在组件外部对组件内部的子组件进行管理的场景。

转发ref到子组件

注意：

  • 只有通过React.forwardRef创建的组件，才有获取和转发ref的能力。普通组件props中不存在ref，组件内部也获取不到ref参数。
  • ref转发既可以用来控制组件内部的dom节点，也可用来控制组件内部的react组件。
