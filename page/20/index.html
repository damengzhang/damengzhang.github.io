高阶组件

发表于 2019-04-19 分类于 web ， react

概念

高阶组件是参数和返回值都为组件的纯函数。

约定

  • 高阶组件需要将与自己无关的props传递给被包裹的组件。

  • 包装显示名称，方便调试

注意事项

  • 不要在render中使用HOC

  • 需要对组件静态方法进行额外复制

  • Refs不会被传递

    props中没有包含ref属性，故不会被传递，可以通过使用 React.forwardRef API解决此问题。
