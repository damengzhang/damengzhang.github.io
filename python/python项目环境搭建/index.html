python 项目环境搭建

1、装所需 python 版本

2、pycharm 配置 python 解释器路径

3、配置 pip 镜像地址（pip.ini放到用户home目录下）

4、pycharm 打开目录从项目根目录打开，requirements.txt 文件会自动被 pycharm 识别

5、按照 pycharm 的提示对所需依赖进行安装
