Hook API

发表于 2019-10-15 分类于 web ， react

基础Hook

useState

useEffect

我们将传递给useEffect的函数称为effect，efffect的作用是执行改变dom、添加监听、设置定时器等操作。

effect的执行时机

默认情况下，effect会在组件被渲染之后执行。与 componentDidMount、componentDidUpdate 不同的是，在浏览器完成布局与绘制之后，传给 useEffect 的函数会延迟调用。

effect的条件执行

可以将其配置成在某些值改变的时候再执行。只需给useEffect传递第二个参数，参数类型为数组，让因变量成为数组元素即可。

如果想执行只运行一次的 effect（仅在组件挂载和卸载时执行），可以传递一个空数组（[]）作为第二个参数

当需要清除effect中添加的监听、定时器等资源时，只需让effect返回一个清理函数，将清理操作放入清理函数中即可。

如果组件多次渲染（通常如此），则在执行下一个 effect 之前，上一个 effect 就已被清除。
