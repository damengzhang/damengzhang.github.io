python不同于其他编程语言的语法记录

发表于 2019-07-23 分类于 Python

1、关键字in的用法

in通常用来判断key是否存在于字典的键，但是python中in还可以用于判断字符串是否包含某个子串

2、代码块的表示

通常代码块用大括号{}包裹，python用缩进幅度来表示所在的代码块

3、列表/字符串截取

list[start:end] 表示截取下标范围[ start, end )的元素
