发表于 2019-07-23 分类于 Python

1、关键字 in 的用法

in通常用来判断 key 是否存在于字典的键，但是 python 中 in 还可以用于判断字符串是否包含某个子串

2、代码块的表示

通常代码块用大括号 {} 包裹，python 用缩进幅度来表示所在的代码块

3、列表/字符串截取

list[start:end] 表示截取下标范围 [ start, end ) 的元素
