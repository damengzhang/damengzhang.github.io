node路径操作

发表于 2018-04-09 更新于 2020-03-08 分类于 web ， node

node.js 使用函数封装器将每个模块进行封装，并将如下参数传递给模块，使的模块可以在内部直接使用这些参数。

当前模块所在路径

__dirname 可以获取到当前模块所在目录。

当前模块文件名

__filename 可以获取到当前模块的绝对文件名。

拼接路径

path.join([...paths]) 方法可以使用平台特定的分隔符，将所给定的参数片段连接在一起。
