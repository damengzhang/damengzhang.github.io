1 imports-loader

1.1 作用

通过 imports-loader 能够给匹配文件自动导入配置模块。比如，对于某个依赖 jquery 的 a.js，通过 imports-loader 导入 a.js 时，webpack 会自动在 a.js 文件中添加 import jquery 语句，而省去了手动导入 jquery 的过程。
