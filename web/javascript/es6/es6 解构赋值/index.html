es6 解构赋值

发表于 2019-03-10 分类于 web ， javascript ， es6

什么是解构赋值

解构赋值是 es6 新增语法，通过解构赋值可以从对象（数组 / 字符串）中将属性值（元素 / 字符）提取出来，赋值或初始化给其他变量。

应用场景

赋值

设置默认值

交换变量

接收函数返回值

将剩余数组赋值给一个变量

注意

解构对象时会查找原型链

如果属性不在对象自身，将从原型链中查找

解构赋值是浅拷贝
