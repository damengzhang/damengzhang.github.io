深入JSX

发表于 2019-04-25 分类于 web ， react

​ JSX其实是React. createElement(component, props, …children)函数的语法糖。故使用jsx前，需先引用React库。

JSX组件标签

  • 大写字母开头的jSX标签才会被识别为React组件

  • 组件标签可以使用点语法

  • 组件标签不能为表达式

JSX中的Props

JSX中的props可以为js表达式、字符串，props属性的默认值为ture，可以将对象使用展开运算符传递给Props。

JSX中的子元素

jsx组件的开始和结束标签之间的内容，会通过Props.children传递给组件。

字符串、表达式、函数作为子元素

组件作为子元素

布尔类型、Null 以及 Undefined作为子元素
